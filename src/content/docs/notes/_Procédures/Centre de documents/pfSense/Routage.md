## 1. Création des routes statiques VLAN

Aller dans **System** → **Routing** → **Static Routes** pour créer les routes vers chaque VLAN.

![image/Attachments 3/Installation\_CISCO\_CSB250\_08012025(1) 17.png](../../../../../../assets/notes/image/attachments-3/installation_cisco_csb250_080120251-17.png)

## 2. Création des gateways

1. Aller dans **System** → **Routing** → **Gateways**
2. Créer les gateways pour les interfaces **LAN** et **WAN**

![image/Attachments 3/Installation\_CISCO\_CSB250\_08012025(1) 18.png](../../../../../../assets/notes/image/attachments-3/installation_cisco_csb250_080120251-18.png)

:::caution
